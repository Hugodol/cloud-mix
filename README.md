## Cloud-Mix
  
<b>This app is a platform for online musicians, songwriters, and producers to come together and mash up a quick jam session to create a song communaly.</b>

<b>Technologies</b>
- Node.js
- React.js
- React Router
- Amazon S3
- MongoDB
- Mongoose
- Dotenv
- Facebook OAuth Passport
- Elasticsearch
- React-Bootstrap
- Wad.js
- Howler.js


<b>Collections</b>
- Users & Songs
